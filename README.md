# Sudoku

Um aplicativo Flutter para jogar Sudoku.

## Funcionalidades

- Modo escuro e claro: Toque no ícone no app bar para alternar.
- Novo Jogo: Gera um novo puzzle aleatório.
- Reiniciar: Reseta o jogo atual.
- Desfazer: Desfaz o último movimento.
- Congelar: Quando uma linha, coluna ou quadrado 3x3 estiver correto, ela fica congelada e não pode ser editada.

## Como executar

1. Certifique-se de ter o Flutter instalado.
2. Execute `flutter pub get` para instalar dependências.
3. Execute `flutter run` para iniciar o app.
